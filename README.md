## Overview

Banking-ATM is a Java-based simulation tool that models essential banking operations through an interactive command-line interface. Designed for educational purposes, it demonstrates core object-oriented principles such as user management, transaction tracking, and secure account handling. This project was created for my Intro to Java class for Butler Community College
